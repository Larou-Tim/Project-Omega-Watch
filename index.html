Help

Search for any pokemon by name or by number.
A color palette for the chosen pokemon will be generated.
Click magnifying glass to select pokemon.
Drag and drop from swatches to containers for element selection.
Once satisfied, you may download the set colors as a style sheet to be used in your own HTML pages.
